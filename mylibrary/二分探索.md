# 二分探索の説明
#define bsearch(a,x) distance(a.begin(), lower_bound(all(a), x))

lower_boundはx以上で最初に現れる位置
- x < b[0] -> 0
- x == b[i] -> i
- b[i] < x < b[i+1] -> i+1
- b[m-1] < x -> m

upper_boundはxより大きい最初の位置
- x < b[0] -> 0
- x == b[i] -> iより右（同じ値を全部飛ばす）
- b[i] < x < b[i+1] -> i+1
- b[m-1] < x -> m(=end)
